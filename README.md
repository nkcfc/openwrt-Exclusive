## 固件说明：

lean源码版本，内核为***5.4***版、***5.10***版、***5.15***、***6.0***版

支持IPV6


自动每5天更新一次，手动不定时更新

> `管理ip：192.168.1.1 密码：password`

### 固件分区默认16M+1G,分区一致可直接web升级，否则请使用DD写盘或重新写盘，首次刷入不建议保留配置，以免发生BUG。
